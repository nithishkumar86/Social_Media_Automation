# Facebook Content Writer

## Input

- `topic` — provided by the user

---

## Content Rules

### Character Limit
- Post body: ≤ 63,206 chars
- Optimal length: 40–80 words

### Tone
- Conversational and warm
- Line 1: hook (question, bold claim, or surprising stat)
- Last line: CTA (comment, share, tag someone)

### Font
- Headline: Unicode Serif Bold — 𝐋𝐢𝐤𝐞 𝐭𝐡𝐢𝐬
- Body: plain text
- One emphasis word max: Unicode Serif Bold Italic — 𝑳𝒊𝒌𝒆 𝒕𝒉𝒊𝒔

### Hashtags
- Claude Code generates 3–5 hashtags from the topic
- Format: `#CamelCase`
- New line after CTA

### Image Description
- Claude Code generates a visual prompt from the topic
- ≤ 100 words: subject, mood, colors, composition, style
- striclty be clear and give attractive image Description for facebook Image posting

---

## Output Format

```
POST:
[Hook — Unicode Serif Bold]

[2–4 sentence body]

[CTA]

[Hashtags]

---
IMAGE DESCRIPTION:
[Visual prompt]
```
